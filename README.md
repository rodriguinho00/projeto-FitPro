# FitPro Academia - Sistema de Gestão

Este é um sistema web para gestão de academias desenvolvido para a disciplina de Programação para Internet na Fatec Jahu.

## 🚀 O que o sistema faz:
- Cadastro e listagem de Alunos (com CPF/RG)
- Cadastro e listagem de Funcionários
- Controle de Pagamentos (Financeiro)
- Grade de Aulas e Horários
- Sistema de Login e Cadastro

## 🛠️ Tecnologias:
- Python (Flask)
- HTML, CSS e JavaScript
- Bootstrap 5

## 📖 Como rodar:
1. Instale o Flask: `pip install flask`
2. Rode o arquivo: `python app.py`
3. Acesse: `http://127.0.0.1:5000`
